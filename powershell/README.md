# PowerShell Configuration

This directory contains configuration files for customizing the PowerShell environment. Below is a description of the files included:

- **Microsoft.PowerShell_profile.ps1**: This file is automatically loaded by PowerShell at startup. It is used to define custom aliases, functions, and other environment settings.

- **files-Init.ps1**: A script for initializing file-related configurations or tasks.

- **functions-alias.ps1**: Contains custom functions and aliases to enhance productivity in the PowerShell environment.

## Usage

To use these configurations, ensure that the files are placed in the appropriate PowerShell profile directory or sourced manually in your PowerShell session.
